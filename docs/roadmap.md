# Roadmap du projet

## État actuel

Le projet permet aujourd’hui :

* la collecte automatisée d’offres,
* le filtrage simple,
* le scoring,
* le stockage,
* la notification.

L’objectif actuel est la robustesse.

---

## Évolutions court terme

Les prochaines améliorations prévues :

* enrichir les filtres métier,
* ajouter de nouvelles sources,
* améliorer la lisibilité des résultats,
* renforcer le debug.

---

## Évolutions IA / LLM

Le projet pourra progressivement intégrer :

* classification intelligente des offres,
* résumé automatique,
* détection des vraies missions backend,
* priorisation des annonces selon profil cible.

---

## Évolutions produit

À moyen terme :

* dashboard simple,
* historique des tendances,
* API enrichie,
* pilotage multi-profils.

---

## Vision long terme

Construire un assistant intelligent capable de :

* surveiller le marché,
* comprendre les annonces,
* expliquer pourquoi une offre mérite attention.
